# 킥보드 통합 플랫폼 - PEOCH

> React + Spring 기반의 안전하고 편리한 킥보드 이용 서비스

## 🛠 기술 스택

- Frontend: React, TailwindCSS
- Backend: Spring, MySQL
- Deployment: AWS EC2, Nginx

## 🌟 주요 기능

- 🚲 킥보드 위치 및 주행 경로 조회
- 📊 사용자 통계 기반 리포트
- 📡 실시간 채팅 (WebSocket 기반)
